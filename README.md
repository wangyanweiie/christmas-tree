# 🎄 3D 圣诞树互动网页

一个精美的 3D 圣诞树互动展示项目，支持照片上传、背景音乐、摄像头互动等丰富功能。

## ✨ 功能特性

- 🎄 精美的 3D 圣诞树渲染效果
- 📷 支持上传自定义照片作为装饰
- 🎵 背景音乐播放（支持自定义音乐）
- 📹 摄像头实时预览互动
- 🎨 多种主题配色（冰雪、金色、经典红等）
- ⚙️ 丰富的自定义设置选项
- 📱 完美适配移动端和桌面端

## 📁 项目文件

```
├── 圣诞树升级版v3.0.html      # 主程序文件
├── blocky_photo_studio_1k.hdr  # HDR 环境贴图
├── christmas-tree.zip          # 圣诞树 3D 模型资源
├── *.mp3                       # 背景音乐文件
└── README.md                   # 说明文档
```

## 🚀 使用方法

1. 直接用浏览器打开 `圣诞树升级版v3.0.html` 文件
2. 等待资源加载完成
3. 使用界面上的按钮进行交互：
   - 上传照片装饰圣诞树
   - 切换背景音乐
   - 调整主题配色
   - 开启摄像头互动

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| 鼠标拖拽 | 旋转视角 |
| 滚轮 | 缩放 |
